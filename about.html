About This Project

NOTE: This project was created as part of a college assignment. It is not intended for commerical use.

Chat with Alex is a web application built using the API.ai service. The chat app was created to provide helpful and quick access to resources by just asking Alex, the bot. The aim of this web app is to provide a simple and easy to use way for finding local mental health services or even as a way to access information or links to mental health material online. It not only helps find these resources but will also chat with you in a natural way and help you in any way possible.

The amount of information Alex holds is limited however, and Alex will do the best possible job helping to find exactly what you need or even just as a friend you can talk to and try out some different questions to test Alex's abilities. Give alex a try and see exactly how helpful the service can be.
